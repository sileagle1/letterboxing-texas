From: Silver Eagle (sileagle@alltel.net) | Date: 2004-08-14 15:08:15 UTC
I agree that March 19 would be better as far as weather goes
and that seems to be the majority choice. Right now there
are 46 campsites still available for that date so it should
work if I can get one of the covered picnic areas. I guess
we can wait a few more days to see if that date is still the
favorite and I will try to call and see when picnic areas
are available.
Several people who were not able to attend last year's Event
have asked what to expect, and my response is that we can
do it any way we want and I am open to suggestions. Last
year we all brought some food and/or the means to eat it
and spent quite a bit of time eating and meeting each other.
Then groups of people went searching for existing or new
letterboxes, and some came back to eat and visit again.
There was no formal schedule of events or itinerary and I
kind of liked the way it went. People just came and went
as their schedule allowed. Some came the night before and
camped both nights (I plan to do this), others just came for
the day. You can go to my Event page to read more about it:

http://www.geocities.com/sileagle1/Events/FTLE/

We can decide how to run the event later, after we pick
a date and get reservations.
Information about Brazos Bend SP can be found at:

http://www.tpwd.state.tx.us/park/brazos/

Besides fishing, hiking and alligator viewing, it also has
the George Observatory, which allows the public to look through
several telescopes on Saturday nights. I highly recommend
doing this and just hope it is a clear night when we are there.
More information about the George Observatory can be found at:

http://www.hmns.org/see_do/george_observatory.asp

Thanks for the input so far and sorry for you letterboxers
that live so far away having to either drive forever or miss
the event. Maybe next year we will have enough letterboxers to
have 2 Events in different parts of the State.

Silver Eagle
