Choi, the main webmaster for LBNA, has created a website that allows letterboxers to submit ideas for improving LBNA.  It also has a place to submit a support question if you are having problems.  I assume the LBNA Advisory Group will discuss these suggestions, so check out what has been submitted and add anything you think should be considered.  However, don't just add something because AQ has it.  Decide if it is something that will make you want to use LBNA more.  Thanks.
