For anyone attending the Great Book Adventure gathering, if you are interested in a copy of the very clever Wild West CLUE Game booklet created by Brandi of the BusyBeesFamily, please check out the file section of our group home page.

http://groups.yahoo.com/group/letterboxingtexas/files/

 

Each letterboxer playing the CLUE game stamped their weapon, suspect and location stamps into these booklets, and we all ended up with very cute souvenirs of the game and the gathering.

These sheets will only be available for a limited time, and then I’ll remove them again from the files to make room for current stuff. (Make sure you check out Brandi’s instructions – I found them invaluable to be able to assemble the booklet!)
