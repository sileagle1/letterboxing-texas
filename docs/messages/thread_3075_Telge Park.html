Telge Park

From: Boots Tex and Lone Star Quilter (janet@redbarnretreat.com) | Date: 2007-09-11 18:27:27 UTC
We just got a report from a new letterboxer about LSQ's box in Telge
Park. They stated that they couldn't get to the box location because
there was water on the path and they couldn't get across.
Nevertheless, they posted the box as "attempted" on LbNA. We went
into the box site and noticed that a letterboxer named Wildflower had
also reported the box as "attempted" and put the date as "0 2020",
which means that report will always show as the latest information (in
this case an attempt) about that box. We don't actually think that box
is missing as it has been found recently by an experienced and
reliable letterboxer. We'll check it out soon. We looked at other
boxes in that park and noticed that this placer had put that same date
on some of them (some found and some attempted). As everone knows, a
report of "attempted" can mean death to a box. We should make new
boxers aware that if they don't actually get to the correct location
of the box, it should not be reported as "attempted". Is there any
way that the date "0 2020" can be removed from a listing?

BT and LSQ
