Hello All,

 

I'm fairly new and introduced myself recently.  I'm enjoying this list and hope some of you can help check out my listing on Atlas Quest.  I've just planted my first box and posted my clue.  I would appreciate it if some of you who are more experienced could take a look at it and let me know if it is up to par or needs any adjustments, are the directions clear, etc.  The name of the box is "Longhorns" and it's the only one in Commerce so far.  I plan to set a couple more to make the trip here worthwhile!  There are geocaches here, but the college kids have evidently not discovered Letterboxing yet, in fact, no one I've spoken with has!  If it weren't for jb kokopelli, we would have to drive a long way to find any boxes!  I'm so excited about planting this box and hope it will have some visitors.  I also included a First Finders certificate.  I appreciate any and all suggestions.
