I've been using a Garmin Extrex (the blue one) for almost 2 years soon.
I've interfaced it to my laptop (for PhotoMapUSA) and I interfaced it to my IPAQ Pocket PC
to use as a driving navigation system in the car.
It has my recommendation!  Get the combo Lighter Adapter/serial data (rs232) cable for it! 
 
The BLUE one is the E-Trex Legend - which has been discontinued.  The next one up is the Venture.
 
The E-Trex series has some serious flaws - the greatest being the patch antenna.  If you are not going to use it anywhere except in an open area, it is OK - just keep it flat so the antenna can see the sky.  Even out here in the desert my E-Trex sometimes lost lock.  (YES - ALL THREE of the E-Trex series - it wasn't a bad one)!  If you use it in the forest, just realize it WILL lose lock.
 
Any GPS with the quad-helix antenna is a much better choice.  I have tried them side-by-side, so I know it is true.  Being a ham operator - I have access to much antenna data.  The Quad-helix is known to be a fine antenna for satellite work.  And after all, what is GPS all about if it isn't satellites?
 
Not saying it won't work.  But the newer ones (as usual) will far outperform the older ones.
 
Also - the GPSMAP 60CS will connect to a standard USB port (still does serial too - as if you would need it...).  The E-Trex is limited to the serial port.  I wouldn't try loading maps on the serial port - it takes all night.
 
Mike.
