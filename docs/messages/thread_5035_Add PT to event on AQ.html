From: Barefoot Lucy (barefootlucy@gmail.com) | Date: 2012-06-02 21:31:39 UTC-05:00

A personal traveler won't be listed under event box. An event box is more like a plain 'ol box that is only "hidden" at the event - sometimes out in plain sight.

A personal traveler, on the other hand, is personal to you.

Here's a trick to see if it is showing up. Go to the event page and click on Print Checklist. It will show you all the travelers you haven't yet collected, including your own.

Lucy
