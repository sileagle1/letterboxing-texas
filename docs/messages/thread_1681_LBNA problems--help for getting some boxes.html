LBNA problems--help for getting some boxes

From: dewberrylb (dewberrylb@gmail.com) | Date: 2006-02-10 13:04:38 UTC
LBNA is having major problems. Choi, one of the webmasters, says it
should be up by noon.

However, if a clue is cross-listed on AQ (even if it is hosted on
LBNA), you can get a version of it.

Just go to the box on Atlasquest.com and after you click on the box
(and the error message comes up), there is a link in the right corner
that says get cached clue. If you click that, AQ's last cached
version will come up.

Just in case you're getting a jump on the weekend and need a clue fix.

-DB
