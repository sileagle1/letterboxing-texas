I had gotten several reports that this box was missing, so I created a new box (Ryan Express 2) and hid it in a new location in Alvin today. Then I went to the old location to see if the box really was gone and sure enough, I found it! It was a little damp but in pretty good shape for a box over 6 years old, so I left it where it was but added a rock on top to help locate it under all the pampas branches. Bottom line, there are now 2 reasons to come to Alvin and pay homage to Nolan Ryan.
