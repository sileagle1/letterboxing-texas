entire park was under water. I was going to place a
box in Millie Bush Park, but nothing was visible but

that when I do place this box, it is going to be about
waist high in a tree!

I found Ode to Soccer in a different part of the park,
and, miraculously, this box was accessible, dry, and
doing well.
