Okay, this is your work-around, until the site is back up.

1. Go to the Texas letterboxing atlas
http://birdsofafeather.netfirms.com/cgi-bin/lbatlas.cgi

2. Look up and note what boxes you might want clues for.

3. Go to Google. type this into the search engine.
site:www.letterboxing.org "name of box"
for example site:www.letterboxing.org "rabbit house" you can also
search by placer name, by county, etc.

4. You should get some hits. Click on the cached link instead of the
name of the box.

It doesn't work for every box, but it works for many of them. You
also may need to try different variations. For instance this search:
site:www.letterboxing.org gramatrick gets you access to different
boxes, but I could never get our pumpkin pi box up for instance
(although you can get those clues by following my previous post about
atlasquest).

Confused? Hope it helps, at least some.
