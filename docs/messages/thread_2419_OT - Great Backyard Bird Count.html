I know quite a few letterboxers are also birdwatchers, so I wanted to mention this now so you can plan ahead a bit, and so those of us who are incidental birdwatchers might make note so we can try it out more diligently!
 
The Great Backyard Bird Count is coming up in February - the 16th to 19th to be exact.  It's not too soon to start planning this, since you will have more fun at it if you do a little research to find out what bird species you are likely to see and to learn more about common ones so you can count more easily.  The website for the GBBC is www.birdsources.org/gbbc
