From: Christine Borgfeld (christine@borgfeld.net) | Date: 2016-12-15 15:12:48 UTC-06:00

Thanks for the offer!  We will certainly need help setting up for the event and managing activities.  We have a core planning team which met in Austin last weekend, made preliminary plans, and put together the announcements this week.  For now, here are some things for everyone to consider in the short term…

 

NLW and Wry Me are planning decorations for the event.  They need donations of Texas themed items for decorations which you can bring to TALE.  We will publish more details and a list of specific needs before TALE.

 

PI Joe and Mosaic Butterfly will be coordinating all letterboxes planted for the event.  If you would like to make a series or box for this event, please start thinking about it now! We will print bound clue books before the event for everyone who registers, so boxes need to be planned well in advance.  We also expect to have 200+ boxes and need to make sure they are not all on the same trail!  Appropriate themes are anything related to Texas, wild west, letterboxing & its history, England/Dartmoor, puzzles/ciphers etc.  We will post donation requests by theme next year, but if you have an idea before then feel free to run it by us.    

 

Regards,

Open Space
