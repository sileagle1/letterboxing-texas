I'm new to the whole palm thing too, so I don't know how much help I
can be, but here it goes...
I have a Palm IIIx ($30 off Ebay!) It has 8mb of RAM which I can
assure you is much more than enough. It runs off palm OS version
3.3.

It came with Palm Desktop software, which is basically every
application that you have on your palm, but on your computer. In the
Memo application I made a separate category for each region for
orginazational purposes.
Then I went online and did a copy and paste from each clue from the
LBNA site to microsoft word. I then edited the text so each clue
was in the same format. Once that was done, I once again copied
that text and then pasted it into correct region in the Memo
application in the Palm Desktop program. All this was done one clue
at a time so I wouldn't go insane.

When I finished all that, I performed a HotSync between the computer
and PDA so the info would show up in the PDA.

I know there are programs out there that make it easier to do than
all that, but those cost money and I'm pretty cheap. The clue
database at LBworld.com will soon be set up so that you can
instantly download clues to your pda without all the copying and
pasting. I think it works by magic.

When I was shopping around I heard good things about the Handspring.
It's recommended by a lot of those tech-oriented geocachers.
Basically all they said was that it should have at least 8megs of
memory and be able to hotsync. I know nothing about the Toshiba, but
I think your handspring should be capable of all that.

Hope this helped,
-MonkeyToes
