Replaced box in Humble

From: Boots Tex (redbarn@eastex.net) | Date: 2013-04-19 21:57:45 UTC
My Governor Series box located in Humble (An Humble Man) was reported missing by Wry Me in January. Today, I had an opportunity to replace it, so I did. It was great weather out there and no mosquitoes! Anyway, I have a question before I post the new clues: The box was indeed missing, but the original location seemed better to me than any other place I could find, so I left the new box there. The stamp, however, is a new and different carve than the original. Same subject, different photo. I'm always torn whether to make this a new box or to keep the old box name and update the clues. Certain details are different, like the trail sign, which was referenced, has been replaced with different wording and a large leaning tree which was there is completely gone, so the box location would be difficult to find using the original clues, although Wry Me did find it. Any opinions? The box is listed as a mystery box only to the hide the name of the park, which was at one time suspected of being unfriendly to letterboxes. The box also has a blue diamond on AQ, which might be a reason to keep the same box name. Of course, that might also be a good reason to make it a different box, since the stamp probably had something to with getting a blue diamond. You can tell I'm confused.

Boots Tex

From: sileagle1 (sileagle@windstream.net) | Date: 2013-04-20 13:11:59 UTC
I agree, new image = new box. The hard part is deciding whether to create a new image or just recreate the old one. I used to replace all my boxes in the old days, but now I only do it if the box is special to me or is really old and I want to preserve its history. One down side to replacing, especially if you change the location, is that the new clues will not be noticed by people who looked for it and did not find it the first time unless they happen to click on it and notice the change. The best thing about you creating a new box is that we now have another Boots box to find in a great park!
