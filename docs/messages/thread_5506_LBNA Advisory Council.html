From: (sileagle1@yahoo.com) | Date: 2015-08-28 14:11:20 UTC-07:00

This group was formed to help Choi (LBNA owner and webmaster) improve the LBNA website and I am privileged to be part of it.  We had our first conference call last weekend to determine our goals and two main ones emerged:  improve LBNA enough to make letterboxers want to use it again; inform new letterboxers of its existence.


We then talked about why so many letterboxers quit using LBNA and the main reasons we came up with were:  the website is unreliable; the search process was cumbersome and did not work correctly; many felt that most boxes on LBNA were missing and clues outdated.  To correct these deficiencies, Choi has already moved the website to a new server, which seems to be working great.  He then fixed the search process and added more functionality to it.  He also added a feature to automatically archive a box if 3 attempts are logged in a row without a find to eliminate the clutter of missing boxes.  A few of us were not comfortable with this solution but he assured us the owner could unarchive the box any time.  To help with this I suggested he set up an email notification to the owner whenever their box is archived so they are aware of it.


To help inform new and veteran letterboxers of the “new” LBNA, we proposed having a contest to see who could find the oldest box.  The details have not been worked out yet and I’m not sure how we would tell people about it without publishing something on AQ.  I’m told there is a new forum on AQ called LBNA Chatter, so maybe that could be our voice.


Choi has made many more changes to the site, which you can see by clicking on the Help tab and then clicking Change Log.  He has also changed the look of LBNA, putting more information on the main page.  If you want the old look you can have that by clicking the Members tab and then clicking My Preferences and choosing Home Style Page at bottom.  So please go to letterboxing.org and look around (be sure to login so you see all the features), then let me know what you think or what you would like to see added or changed.  You can reply publicly to this group so we can all discuss or to me privately.  If you have suggestions for Choi you can click on the icon in the lower right corner of the website to send it directly to him.   We plan to meet again next month and I would love to have your feedback before then.  Thanks!

I think Choi has done a great job with the new LbNA site and it seems to be working perfectly.  I agree that something needs to be done about weeding out some boxes that are missing so that a lot of time isn't wasted hunting for a box that doesn't exist.  What I find is that most finders are reluctant to post an "attempt", so it's fairly unusual for a box to accumulate 3 attempts in a row.  What I usually see is a bunch of finds, then suddenly none at all. so it's hard to tell when a box is missing.  If an attempt is posted, it seems that no one ever looks for the box again, so the details will show a lot of finds, then one attempt and then nothing after that.  We should probably be, as planters, a little more careful to either retire or archive boxes that we know are missing.  Let me ask this question:  does archiving or retiring a box affect a planters box count?  In other words, if I have planted 100 boxes, does it change my plant count to 99 if I archive or retire a box?  Just wondering.

I agree that it is frustrating how many boxes are probably missing but are still listed, including many owned by me.  However, I am usually loathe to archive a box based on one person not finding it, and like you said, rarely does another person look for it once it has an attempt.  Since many of my boxes are too far away to check on personally, I can only rely on reports from other boxers, which can be wrong.  So I don't see a good way to resolve this issue, but we can try to clean up our boxes as best we can.
