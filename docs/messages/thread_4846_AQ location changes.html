AQ location changes

From: Christine Borgfeld (christine@borgfeld.net) | Date: 2011-07-16 00:22:59 UTC-05:00

In the last few days there have been some big changes on AQ regarding how the website handles a box’s location.  There are several new features with the change including custom locations (which is very cool!).  You can see the details here:  http://blog.atlasquest.com/

 

Unfortunately it looks like there are some glitches.  I’ve seen some strange things in search results and one of my box locations changed.  It used to have the park in the location and now it’s just showing the center of the city.   You may want to look through the locations on your AQ box listings and see if anything strange jumps out.
