From: Jenifer Barbee (jaydawnlove@yahoo.com) | Date: 2010-11-29 18:13:06 UTC-08:00
Treasure Hiking is letterboxing with hiking. You are hiking to find letterboxes thus the name Treasure Hiking. TH points are the miles you hike. TH stands for Treasure Hiking. So if you hiked a mile to get to a box then you have one TH point. 25 points gets you a path tag PI Joe had made for us. You can see the pathtag on his newest plants clue page  http://www.atlasquest.com/showinfo.html?gBoxId=161704. This box series will get you 3.5 points. There are rules like you can't count the mileage while lost. You must only count the miles to the box and back as the clues read. And you may count miles you hike to plant a box but not to maintain the box. I'm sure someone can point you to the website that explains it all. But that's the just of it...
 
~Dawns Glory~ with over 78 TH points...

From: Lou Catozzi (lou.catozzi@att.net) | Date: 2010-11-29 20:00:49 UTC-08:00
The Treasure Hiking program was created to encourage letterboxers to go after some of the more remote boxes around the country. It works by awarding points for each box that requires at least a 1.0 mile hike to locate and return to the starting point of the clues. The points for a box are equal to distance hiked in miles. Fractional miles all count if the total hiked for a given box is over 1.0 miles so a box 1.4 miles from the clue's starting point would be worth 2.8 points. Points are awarded for box finds and box plants but not for "not founds". There is a widget available on AQ that tracks TH points for you. When a boxer has reached certain milestones they can buy a patch or PathTag, a quarter-sized metal coin, to commemorate the achievement. In Texas we currently have 25-Mile PathTags available for $3. You can see the design of the PathTag at:
http://www.atlasquest.com/showinfo.html?gBoxId=161599
