From: (blhjrh@aol.com) | Date: 2008-07-20 13:18:57 UTC-04:00
I've had some finders report that my Mystery box "Simple Gifts: Music Box #10" is missing, so I checked on it just the other day while out doing some routine letterbox maintenance and I'm happy to report that it's still quietly and safely hidden just where it's supposed to be.  The mystery clues aren't that difficult to solve, but I think perhaps the reason some finders haven't found it (even after solving the mystery clues) is that it is really well-hidden - very carefully placed so as not to be at all visible or to be accidentally discovered because it is in a fairly vulnerable location.  So, just in case you're one of those who have tried and not found it, that letterbox is still there, I'm happy to say, and hopefully will continue to remain well-hidden (and many many thanks to previous finders who did take such care to replace it exactly where they found it so that it has lasted so well!)
