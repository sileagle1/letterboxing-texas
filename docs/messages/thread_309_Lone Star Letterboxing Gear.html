Lone Star Letterboxing Gear

From: CelticLions (celticlions@comcast.net) | Date: 2004-09-09 23:20:22 UTC-05:00

The dust has settled and we have two XL Lone Star Letterboxing T-shirts
left - I ordered a couple extra, because I couldn't predict how many of
my sons would want one ;-).
If you'd like one or both, please email me off-list:
celticlions@comcast.net

Also, I noticed that we've added several new members to the group -
that's exciting! If anybody would like to get another group order going,
we just have to order a minimum of 12 shirts: the basic price is $9.00,
with larger sizes costing a couple of dollars more. If you're
interested, please mail me off this main group list for details.

(BTW I wore my Letterboxing T-shirt in to talk to someone in a public
building about placing an inside letterbox today - it helped to look
semi-legitimate!)

NOTE: Down the street from the T-shirt silk-screener is a company that
creates embroidered patches. Would anyone be interested in having me
check out prices for a Lone Star Letterboxer patch? I think they'd be
cool to add to backpacks, hats or heavier shirts (or for holiday gifts).
I've never ordered patches from them, but I've done business with them
for years to make pins, so I know they're reliable.

If I can convince my husband or sons, we'll create an order page
someday, so that this can be easier for everybody!

Take care,
Ruby Tuesday
CelticLions
