From: dewberrylb (dewberrylb@gmail.com) | Date: 2005-07-12 23:59:21 UTC
Okay, what a great response! So far, from those who have responded
here and to my email, we have 11 participants. We certainly will
give it some more time, as lots of folks are on vacation or just
don't check their email every day.

I checked into it and there are 16 official state things. I'll list
them and you can see how many you know--test your Texas trivia
knowledge!

Flower
Bird
Tree
Gemstone
Insect
Shell
Pepper
Grass
Motto
Vegetable
Fruit
Dish
Dance
Small mammal
State fish
flying mammal
large mammal

If that's the theme we end up with, I'm more than fine if some folks
want to come up with a tongue-in-cheek version of some of the
above: unofficial candy (has to be red hots), unofficial state
reptile, (???), unofficial stinging insect, etc. Will wait for a
while, see if inspiration strikes anyone, and then set up a poll
with the ideas voiced so far and participants (and potential
finders) can vote. Feel free to post your ideas here or send them
to my gmail account.

Later on, I'll ask participants to send me their addresses, and I'll
set up a lottery for box distribution. Once you know who you'll
send to, you can see if they have any specific requirements, like
Miss Moon's for a micro (how fun to have a box in Gotham). The
lottery will be rigged, so that there is a good geographic
distribution. The lottery holder can be bribed, and will take
requests if doable.

If you want to see a bit more, you can check out spring chick's
site: stampinthebox.com. The box exchange is under projects.

Fun, fun, fun. Thanks for your enthusiasm.

DB
