Note to Rusk gathering attendees about Big Bertha's

From: Barefoot Lucy (barefootlucy@gmail.com) | Date: 2006-06-24 08:59:11 UTC-05:00
I thought it would be appropriate to tell y'all that, sadly, Big Bertha's has closed.  We went to eat there a few weeks ago and it was sealed up tight!  I heard a rumor that they had re-opened, but apparently it was not to be since I tried to eat there the other day and it was closed then too.
 
I have to wonder if we gave them false hope with that huge Friday crowd, and they've been wondering since then where all those customers disappeared to!  That was a fun evening to be sure...
