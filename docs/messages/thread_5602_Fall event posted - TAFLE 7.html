Speaking of events…. I posted the details for TAFLE 7 on Atlas Quest yesterday.  The main event will be Saturday October 21 at Palmetto state park.  Plans for Friday will be announced later. 

https://www.atlasquest.com/showinfo.html?eventId=3837

 

Please note that tent campsites are very limited at this park, so I reserved 5 sites for us to share if you want to tent camp.  We can have 8 people per site so hopefully that will be enough.  If you want a site for an RV or trailer, please make reservations ASAP.  The website says that the sites with electricity are for RV or Trailers only – no tent only camping there.

 

Also the parks have a new rule…. For any group facility rental we must submit a list of first and last names of everyone who will be using the facility.  I’ll need to put this together 2 weeks before the event so please be aware I’ll be contacting you for this before the event.  Sorry for the inconvenience - I’ve already expressed my frustration to them over the new requirement.
