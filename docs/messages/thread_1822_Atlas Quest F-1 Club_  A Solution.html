Since you can't log finds on boxes unlisted in AQ unless you are a
premium member, I have created a bogus box in AQ called F-1 that
anyone can log as a find with any date they choose. This should
allow anyone to be able to join the F-1 club and see those clues.
Mike, give it a try.
