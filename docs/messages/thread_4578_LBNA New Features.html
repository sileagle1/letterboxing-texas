LBNA New Features

From: sileagle1 (sileagle@windstream.net) | Date: 2010-06-21 02:07:59 UTC
I guess I'm a little slow, but I just noticed some new search features on LBNA that Choi implemented late last year. There is now an option to search by location (city & state, or zip code) which will return up to 50 nearby boxes, showing how many miles away they are. Also, when you look at the clues to a box, there is an icon (circle with dot in center) next to the LBNA # at the top that will show up to 30 nearby boxes, also showing how many miles away they are. Now the reason I did not notice these new features is because they are not enabled by default. You have to go to Member Services and click on Search By Location at the bottom to enable it. I asked Choi why it is not turned on by default and he told me it was a compromise. Many at LBNA were against this feature so the only way he could add it was if he did it this way and did not advertise it. He did give me permission to tell people on my Regional List, so that is what I am doing. I told him I thought it was a nice feature, but suggested he could make it even better by adding an input field for miles to limit the search (e.g. search for all boxes in Sugar Land up to 5 miles away only). Not sure if he will implement it, but I will let you know if he does.

BTW, you can see what changes the webmasters make to LBNA by clicking on the Change Log at the bottom of the LBNA main page.
