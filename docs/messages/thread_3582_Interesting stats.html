Baby Bear,
 
I'm the one building this VERY unofficial list, and it is taking some massaging to account for things like archived boxes.  I had to set it aside for a few days because I had a family thingy I had to work on, but I'm gonna try to get it posted this evening. 
 
One of the biggest problems I've run into is that before we had the basement on LBNA, early LBNA letterboxers like you deleted boxes that were no longer active.  Another good example is Jay Drew, who has, by his record, something close to 400 boxes, but only shows 139 on the database (and that's accounting for basement boxes).
 
I really didn't set out to create this "prolific boxer" list, but after crunching numbers to create the number of boxes by state comparison list earlier this week, I took the same information and started re-sorting it to see how many boxes really prolific boxers have - just out of my own curiosity.  It got interesting so I kept on...and then I was in way too deep to back up, so I now have to see it through.  Please though - no one go mortgaging their signature stamps based on the results of my little list - it will almost certainly be arguable and challengeable and even debatable and for sure not scrupulously accurate!
 
Lucy
 
P.S.  You're fairly high on the list as I recall...
