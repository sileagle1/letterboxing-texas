Yes, having a list in spreadsheet format is very handy! We always
create a customized list when we go on a trip. We delete the state
column, add columns for time estimate/distance & comments, then
organize it in a sequence we plan to look for them and print it out.
After we find a box (or don't find it), we use the comment column to
record any notes about the box status that we may want to pass along
to the placer when we get back home. When we do get back home, we re-
sort the list by Placer, so we can easily see all the boxes we found
by a particular placer and then send a status report to them
regarding all their boxes in a single message.
