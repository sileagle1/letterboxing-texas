Hi there!

My attention has been diverted for a couple of weeks
due to scouts, so I'm getting back to the barnraising
this week.

Here's what we need to be doing:

Making your boxes based on information in the counties
we will be serving. I've posted quite a bit on
Johnson and Ellis and hope to get Navarro covered in
the next couple of days. You can also do your own
research on those counties if you'd like.

Yes, we will be doing a potluck. However, I'm still
working out the details on where we'll gather up -
Waxahachie hasn't been real fast in getting back to me
about the park I would like to use and one of my
biggest questions is "how much power we will have".
That said, it may need to be a cold pot luck.

I would like to start getting a headcount if I can
please!

Here's the link until I have a chance to get it
posted. If someone else wants to post it on my
behalf, that'd be fine. I'm guessing it will be this
evening for me to get to it - if I didn't have this
J-O-B thing hanging over my head every day, I'd have a
lot more time for letterboxing stuff!

Anyway, thanks for your questions and if you have any
more, shoot 'em out there like you did. I don't keep
a real close watch on this account during the day, but
I do check it periodically if time allows.

Lucy
