From: (sileagle1@yahoo.com) | Date: 2015-08-31 06:19:44 UTC-07:00

If you have visited LBNA you have seen the new displays on the main page, including Most Recently Listed, Tip Of The Day, and Most Recommended Clues.  The last one comes from the rating buttons on the status page for each box, which were added to allow finders to "rate" a box.  Nobody will be able to see the rating (not even the owner) or who gave it, and no Box Of The Month award is given.  It is just used internally to show which boxes people most recommend others to find.  Eventually it will be added as a Search feature when more boxes have been rated.  Some of you must have already been using it since Baby Bear has several listed.  You can read more about it in this article:
