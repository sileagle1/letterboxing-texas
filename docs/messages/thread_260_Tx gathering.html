   Date: Wed, 11 Aug 2004 18:27:46 -0000
   From: "Silver Eagle" <sileagle@alltel.net>
Subject: Re: TX Gathering? - My Idea

I was planning on trying to have another Event next spring
at about the same time, since it seemed to work out pretty
well with most people's schedules.  I picked Bastrop last
year since it was fairly Central, but I was leaning towards
going to Brazos Bend SP this time (west of Houston).  I know
it would be farther for some to drive, but I thought it would
be nice to have it somewhere different every year, and that
is a pretty good place for the Southeast Region.  I was even
thinking it could have a Rodeo theme, since the Houston
Rodeo will be going on at that time.  Something like
Texas Letterboxing Rodeo Event.  I have not done anything
yet, like check the park for a place to meet, so if
anyone else wants to do something different, that would
be fine with me.  I'm not sure Fall would work since soccer
and other kid sports last until Thanksgiving, then you
have the holidays, and colder weather makes camping iffy.

Silver Eagle
