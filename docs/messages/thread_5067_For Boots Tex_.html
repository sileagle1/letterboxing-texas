Hey...

A long time ago we passed some info back and forth about building radios.
Primarily Xtal sets IIRC, but Heathkit was discussed too. So I wanted to
share this with you. I just bought a radio kit for $250. It is a ham
radio, and it covers the 40m, 30m, 20m, 17m and 15m bands. It is Morse code
only, has 5 watts out, and weighs 5 ounces. It can fit in a good shirt
pocket.

http://www.cqdx.ru/ham/qro-qrp/kd1jv-qrp-ats-4b-5-band-cw-rig-with-psk-mode-
capabilities/

Also I picked up a set of Heathkit twins from 1959. The MR-1 Comanche
Receiver, and the MT-1 Cheyenne Transmitter. I have the receiver working
fine already. The transmitter has some problems, but nothing I can't fix.

http://www.rigpix.com/heathkit/mr1.htm Transmitter looks the same.
