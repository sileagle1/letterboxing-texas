Anyone thinking of coming to El Paso (please do) for a letterboxing Adventure,
contact me. I can give you some information that will not only help you plan
the trip, but also save you some $$$$. If you visit one park first, that fee
can apply to the other parks. It doesn't work the other way around, so the
order in which you search can cost you a few $$$ less if you plan it right.

And remember - to search for Hueco Tanks #1, you need to attend a park
orientation briefing and make reservations weeks ahead of time. But it is worth
it - ask Silver Eagle. The letterbox isn't anything spectacular. But the
park......
