From: (bluebutterflytexas@yahoo.com) | Date: 2014-09-26 14:34:27 UTC-07:00

LBNA:
1. Sign in
2. Click on Members
3. My preferences
4. Edit membership info
5. Type in new Trail Name
6. Save changes

AQ:
1. Sign in
2. Hover over "My Page"
3. Choose Account Info
4. Bottom half of page is Trail Name
5. Type in new Trail Name
6. Update

It is my understanding, this only changes your Trail Name.
