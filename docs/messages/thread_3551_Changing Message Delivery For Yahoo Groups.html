Changing Message Delivery For Yahoo Groups

From: Silver Eagle (sileagle@alltel.net) | Date: 2008-05-14 14:25:32 UTC
Recently we have had a few people leave our group because they can't
handle all the emails generated by all the posts to this list. Even
though email is the default message delivery option for Yahoo Groups,
you can change it. First, go to the Yahoo Group and log in:

http://groups.yahoo.com/group/letterboxingtexas/

Click on Edit Membership up near the top. Scroll down to Step 2
Message Delivery options and select Daily Digest or Web Only. Then
click Save Changes near the bottom.

Hope this helps.

Silver Eagle
