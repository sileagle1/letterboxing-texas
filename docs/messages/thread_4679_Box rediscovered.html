Box rediscovered

From: Boots Tex (redbarn@eastex.net) | Date: 2010-12-03 05:53:40 UTC
We're on a short trip to West Texas to visit relatives in our old home town and came prepared to replace my letterbox called "The Home Run That Never Was", which had been reported missing over a year ago. This box was once named Box of the Week by AQ, so I thought it should be replaced. I actually found the original box exactly where it was supposed to be but with the logbook missing. The stamp was intact, which technically made it an active letterbox, right? At any rate, I took the old box and left the new one with a new but similar stamp and a new logbook. I don't think even geocachers would take the logbook and leave the stamp.
