I have reserved the Dining Hall at Kerrville-Schreiner Park in Kerrville for 3/2/19, TX Independence Day, so the theme will be TX Proud.  I thought we would try something different than a State Park next year to see if we can avoid some of the problems from this year.  Open Space had an event there in 2012 and I think it went well, with lots of boxes still in the park on many miles of trails.  You can sign up on AQ:
