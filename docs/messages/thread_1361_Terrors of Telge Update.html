Terrors of Telge Update

From: TeamKing (teamking@entouch.net) | Date: 2005-10-03 16:41:13 UTC-05:00
Just a reminder that we are having a mini-gathering at Telge Park in
Houston on Saturday Oct. 29th from 3-5pm. There will be new Halloween
themed stamps planted in the park as well as some boxes out on the
table. Also an event stamp carved by Gramatrick.

We are not allowed to reserve the park, so we'll try to get a table
as soon as we get there. Hopefully, there won't be a bunch of other
people.

If you like, you can sign up for this event on Atlas Quest. If you'd
like to create a stamp, please let me know what you will be doing so
we don't have duplicates. All boxes should be planted prior to the
event so everyone can bring the clues with them, unless you plan on
bringing copies of your clue for everyone that day.

I will be bringing 3 traveling event boxes:
-The Munsters Traveling Event Box - with all 6 of the characters -
carved by She Who Plants of Tennessee
-my own Tribute to England Event box - with 3 stamps
-Under the Sea - a collection of 10+ sea creatures created by
numerous people - this started out as a PLB (postal letterbox), but
will now be an event box

Also, upon request only, ask me to see the 'TeamKing Pumpkin'. I'll
be carrying this one around so you'll have to ask to get it!

Hope to see you there!

TeamKing
