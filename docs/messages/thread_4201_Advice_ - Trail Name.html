From: Silver Eagle (sileagle@windstream.net) | Date: 2009-07-31 13:21:33 UTC
When I started letterboxing I had recently joined Indian Princesses with my daughter, which required me to come up with an Indian trail name, so I just used it. But to come up with that name, I thought about what things I like in nature (eagle), then some description of myself (silver hair, better than bald). I suggest you do the same, only don't limit it to nature. Or you could do like Baby Bear and
use an old family nick-name, but you might regret it later!

Silver Eagle
