Another thing to consider is stamps that require several different colors. A stamp might be normal size, but have so many details that require different colors that the first ones dry before you are done. I'm not saying stamps like this should be avoided, especially since I have carved several myself, but it does require a speedy, though accurate, inking process with fine-tip markers. Another alternative is to create a stamp with the colored areas carved away, so you stamp with black and color in with markers later. Baby Bear does this quite a bit and it makes every stamp a little different, since nobody colors them in quite the same way.
