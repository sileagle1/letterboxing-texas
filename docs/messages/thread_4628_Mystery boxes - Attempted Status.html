I must admit that I do mark some boxes "attempted" if I'm pretty sure they are not there for the main reason that I mark boxes "found"; it helps me remember which boxes I have already looked for. With so many out there, it is hard to remember which ones I have already found or attempted, so the status helps me. Unfortunately, it can be a negative thing for the box if it really is there since many people will not look for it. Hopefully someone will find it and mark it found after the attempt, or the owner can mark it found when reported.
As long as people take "attempted" literally and not to mean that the box is missing, it should work.

Silver Eagle

A few weeks later, it was found by other letterboxers, so obviously I was not in the right place. It happens. Glad I didn't allow my ego to mark it as attempted.

Re: [letterboxingtexas] Re: Mystery boxes - Attempted Status

From: Tasha Morris (keishkakeishka@yahoo.com) | Date: 2010-09-21 09:12:16 UTC-07:00
I also mark some boxes "attempted" if I'm pretty sure that they aren't there.  I usually only do this if the box is an older box that hasn't had any finds or attempts listed in the past year, and if there is something in the clues that makes me think that something has changed in the area that the box is supposed to be.  I usually mark these types of boxes as "attempted" only if I'm absolutely certain that I have the right spot, based on some sort of unique landmark where the box is supposed to be, and even then, only if the box hasn't been found in a long time.  It helps me keep track of what I've gone after, and I try to always let the planter know what's going on in the area where the box is so that if necessary they can retire it.  If a planter lets me know that the box is indeed there, and I was way off, then I remove my attempt so that I know to try again and so that other folks don't take my attempt as a sign the box is gone.

Rogue Artist

From: Boots Tex (redbarn@eastex.net) | Date: 2010-09-21 16:30:24 UTC
But that's the reason the "attempt" status is available. If someone who has a lot of finds and experience marks one of my boxes "attempted", I respect that and consider the box missing unless proven otherwise. Those folks always include a note explaining why they think the box is missing, then I can put a note in the clues stating that the box has been reported missing. When I'm hunting boxes, I don't let one "attempt" stop me from searching, but two in a row will make me reconsider.
