From: Barefoot Lucy (barefootlucy@gmail.com) | Date: 2008-06-10 15:46:54 UTC-05:00
You make a very good point that not everyone may know of - you can receive notifications of new boxes from both LBNA and AQ.  On LBNA, you sign up for what is essentially another yahoo group, but there is no discussion - only posting about new boxes.  I personally have mine set to digest, so that I get a new message either every day OR every time 25 new boxes queue up. 
 
On AQ, you can do a search of whatever kind you want, then save the search and somewhere in the process it asks if you want to be notified when something meets your saved search's criteria. 
 
If you were signed up on either site, you wouldn't have been surprised to learn that Waco had so many new boxes!
