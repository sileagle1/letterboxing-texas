From: Debbie Kotlarek (kotlarek@wi.rr.com) | Date: 2011-03-09 17:13:16 UTC-06:00

It is a useful feature for us.  After we open it in Excel, we delete several columns, including the actual clue text.  We then use the other info to put them in order of  a route we plan to follow when we are on a trip.  We also ADD columns for approximate time/distance for the box, a column for noting what Find # that box is if we find it, and a column for comments where we note things such as condition of the box, etc.  We put this summary list in the front of a binder and after it we put all the clues in the same order as our master list.
